# RoomFlow

RoomFlow — веб-сервис для управления бронированием переговорных комнат на `Django 5.2`, `PostgreSQL` и `Docker Compose`.

## Возможности

- регистрация, вход и профиль пользователя;
- каталог переговорных с фильтрами;
- карточки комнат с описанием и оборудованием;
- создание, редактирование и отмена своих бронирований;
- защита от пересечения активных бронирований;
- очередь согласования для модераторов;
- административная панель Django;
- адаптивный интерфейс на `Bootstrap 5.3`.

## Быстрый старт

1. Скопируйте `.env.example` в `.env`.
2. Запустите проект:

```bash
docker compose up --build
```

3. Откройте:

- `http://127.0.0.1:8000/`
- `http://127.0.0.1:8000/admin/`

По умолчанию суперпользователь берется из переменных `DJANGO_SUPERUSER_*`.

## Полезные команды

```bash
docker compose exec web python manage.py createsuperuser
docker compose exec web python manage.py seed_demo
docker compose exec web python manage.py test
```

## Архитектура

- `core` — главная страница, общие mixins и утилиты;
- `users` — регистрация, профиль, роли;
- `rooms` — переговорные и оборудование;
- `bookings` — бронирования и бизнес-логика;
- `dashboard` — согласование бронирований.

Подробное описание решений находится в [doc.md](./doc.md).
